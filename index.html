Cómo configurar tu firma en Google Workspace

  1. Busca tu nombre abajo.
  2. Opción A — Gmail (recomendada): haz clic en Abrir firma → en la pestaña nueva selecciona todo (Cmd+A) → copia (Cmd+C) → pega en Gmail.
  3. Opción B — Código HTML: haz clic en Copiar código del cuadro de texto, o selecciónalo manualmente.
  4. Abre Gmail → ⚙ Configuración → Ver toda la configuración → pestaña General → sección Firma.
  5. Crea o edita tu firma y pega con Cmd+V.
  6. Márcala como predeterminada y guarda cambios.

Gmail necesita el contenido renderizado (Opción A). El código HTML del cuadro sirve para respaldo o configuración por administrador.
